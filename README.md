🏥 Hospital Management System

🌻About The Project:

This is a full-stack Hospital Management System web application designed to manage hospital operations such as patient records, doctor management, and appointment scheduling.

The project also demonstrates modern DevOps practices including CI/CD automation, containerization, and cloud deployment.

✨ Features:
👤 Patient Registration & Management
🩺 Doctor Management
📅 Appointment Booking System
🔐 Secure Login Authentication
📊 Admin Dashboard
🌐 Live Deployment on Cloud
🛠️ Tech Stack

Frontend:

HTML • CSS • JavaScript

Backend:

Node.js • Express.js

Database:

MongoDB

DevOps Tools:

Jenkins • Docker • SonarQube

Cloud:

AWS EC2

🚀 Getting Started
🔹 Installation:

git clone https://github.com/tabassum-321/Hospital-Management-System.git

cd Hospital-Management-System

npm install

▶️ Run Locally
npm start

App will run on:
👉 http://localhost:5000

🐳 Run with Docker:

docker build -t hospital-app .

docker run -p 5000:5000 hospital-app

🔄 CI/CD Pipeline

This project uses an automated DevOps pipeline:
Code pushed to GitHub
Jenkins builds the project
SonarQube analyzes code quality
Docker creates container
Deployment on AWS EC2
